## 摘要
低秩适配（LoRA）合并能够高效地结合多个训练好的LoRA在扩散模型中的多样化生成能力。然而，现有的LoRA合并技术常常遭受严重的参数干扰，导致共享参数空间中出现破坏性冲突。为解决这一问题，我们提出了子空间信号路由（SSR），该方法通过路由内部信号而非执行参数空间合并来消除干扰。具体地，SSR首先沿着秩维度拼接候选LoRA以构建统一子空间。接着，SSR利用逆相关矩阵对该空间内的混合信号进行去相关。最后，方向引导矩阵将这些净化后的信号引导至各自的任务特定子空间。我们提供了严格的理论分析，证明SSR符合普通最小二乘法（OLS）解，从而确保了数学上的最优性。我们利用充分统计量的可加性设计了一种流式算法，这使得能够进行实时更新，显著降低了内存开销和计算时间。大量实验验证表明，SSR在保持可比较效率的同时显著优于现有最先进方法。代码可在https://github.com/nagara214/SSR-Merge获取。

## Abstract
Low-Rank Adaptation (LoRA) merging can efficiently combine diverse generative capabilities from multiple trained LoRAs for a diffusion model. However, existing LoRA merging techniques often suffer from severe parameter interference, causing destructive collisions in the shared parameter space. To address this, we propose Subspace Signal Routing (SSR), which resolves interference by routing internal signals instead of performing parameter-space merge. Specifically, SSR first constructs a unified subspace by concatenating candidate LoRAs along the rank dimension. Next, SSR employs an inverse correlation matrix to decorrelate mixed signals within this space. Finally, a directional guide matrix steers these purified signals into their respective task-specific subspaces. We provide a rigorous theoretical analysis proving that SSR aligns with the Ordinary Least Squares (OLS) solution, thereby ensuring mathematical optimality. We utilize the additivity of sufficient statistics to design a streaming algorithm. This enables on-the-fly updates that significantly reduce memory overhead and computation time. Extensive experiments validate that SSR significantly outperforms state-of-the-art methods while maintaining comparable efficiency. Code is available at https://github.com/nagara214/SSR-Merge.

## 1. 核心问题与整体含义（研究动机与背景）

扩散模型通过 LoRA 等参数高效微调技术可以快速适配下游任务，但实际应用中用户常需要将多个不同任务（如不同风格、物体）的 LoRA 合并到一个模型中使用。现有合并方法（如线性平均、Task Arithmetic、TIES、DARE）直接在参数空间进行算术操作或稀疏化处理，导致严重的参数干扰——不同任务的激活信号在共享空间中“串扰”，造成概念稀释、属性错配或特征丢失。作者将模型合并问题重新定义为 **子空间内的信号路由**，而非参数算术，旨在不引入额外推理延迟的前提下消除干扰，保留各任务的全部能力。

## 2. 方法论

### 核心思想
- 将多个 LoRA 的低秩更新视为不同任务在统一低秩子空间中的信号。
- 构建一个 **路由器（Router）R**，它不修改原 LoRA 参数，而是通过统计信息对混合信号进行去相关和导向，使每个任务的输入激活只激活自身的 up-projection 部分，从而消除跨任务干扰。

### 关键技术细节
1. **统一子空间构造**  
   将 K 个 LoRA 的 down-projection 矩阵 $\{A_k\}$ 沿秩维度垂直堆叠得到 $A_{\text{comb}} \in \mathbb{R}^{Kr \times d}$，up-projection 矩阵 $\{B_k\}$ 水平拼接得到 $B_{\text{comb}} \in \mathbb{R}^{d \times Kr}$。此时合并后的更新可写为 $\Delta W = B_{\text{comb}} \cdot R \cdot A_{\text{comb}}$。

2. **路由器 R 的构建（基于二阶统计量）**  
   - **相关矩阵 G**：$G = \sum_{k=1}^K Z_k Z_k^\top$，其中 $Z_k = A_{\text{comb}} X_k$ 为所有任务输入在统一子空间中的投影。$G^{-1}$ 起去相关（白化）作用。  
   - **方向引导矩阵 Q**：$Q = \sum_{k=1}^K E_k (A_k X_k) Z_k^\top$，其中 $E_k$ 为块选择矩阵，将局部理想激活 $A_k X_k$ 与统一投影关联。  
   - **路由器**：$R = Q G^{-1}$。

3. **理论最优性证明**  
   通过几何分解，证明 $R = B_{\text{comb}}^\dagger \cdot \hat{\beta}_{\text{OLS}}$，其中 $\hat{\beta}_{\text{OLS}}$ 是普通最小二乘（OLS）估计量。因此 R 严格最小化 $\sum_k \| B_{\text{comb}} R Z_k - Y_k \|_F^2$，即特征重建误差。

4. **流式算法与结构重参数化**  
   - **流式更新**：依靠充分统计量的可加性，按批次在线累计 $G$ 和 $Q$，无需缓存所有中间激活，内存复杂度降为 $O((Kr)^2)$。  
   - **部署重参数化**：将路由器融合到 up-projection 中：$\tilde{B}_{\text{comb}} = B_{\text{comb}} R$，最终模块形式与标准 LoRA 一致，零推理开销。

5. **校准数据构造**  
   无需真实图像，仅用每个任务的一条文本提示（如 “A [V] dog”），执行单步前向传播提取输入特征 $X_k$，由此构建路由器。理论分析证明估计误差以 $O(\sqrt{Kr/N})$ 上界收敛，有效样本量远大于子空间维度。

## 8. 不足与局限

- **局部最优性**：SSR 优化的是单层的线性重建误差，未在全局非线性扩散过程中保证最优。尽管实验显示局部最优逼近全局上限，但在极端条件下（如严重领域冲突或语义高度重叠）差距可能扩大。
- **高语义重叠任务**：当任务概念非常相似（如同类物体不同变体）时，参数干扰更强，路由可能更困难。论文未专门测试此类极端情况。
- **潜在滥用风险**：高质量多概念组合可能被用于生成欺骗性内容，需负责任的部署。
- **未探讨其他 PEFT 方法**：仅针对 LoRA，虽然 LoRA 最流行，但方法是否可推广至 Adapter、Prefix Tuning 等未验证。
- **未考虑动态任务数量**：当前所有 LoRA 需要预知并一次性合并，不支持在线新增任务而不重新构造。
