## 摘要
近年来，无需训练的图像编辑因其能够利用强大的预训练扩散和流匹配模型对真实图像进行修改而无需额外训练，逐渐受到关注。然而，现有的基于反演和基于差分流的方法通常执行全局潜变量传输，这不可避免地会将编辑效果传播到非目标区域，导致背景泄漏。为解决这一问题，我们提出了SAM-Flow，一种源锚定掩码流框架，用于局部无需训练的图像编辑。SAM-Flow不是更新整个潜变量表示，而是首先使用侦察图像和基于标记的注意力图来定位可编辑的语义区域。然后，它仅在这些区域内应用差分速度更新，同时将剩余区域锚定到源图像的潜变量轨迹上。为进一步提高空间稳定性和边界自然性，我们引入了一种时变源锚定投影机制，包含动态软掩码、过渡区域和时间掩码累积。所提出的方法即插即用，可与主流流匹配骨干网络（如Stable Diffusion 3和FLUX）集成，无需任何微调。大量定性和定量实验表明，SAM-Flow在实现精确语义编辑的同时，显著提高了背景保持能力，为无需训练的图像编辑提供了一种简单且通用的局部编辑范式。代码地址：https://github.com/chwbob/Sam-Flow。

## Abstract
Training-free image editing has recently attracted increasing attention due to its ability to modify real images using powerful pre-trained diffusion and flow-matching models without additional training. However, existing inversion-based and differential-flow-based methods usually perform global latent transport, which inevitably propagates editing effects to non-target regions and leads to background leakage. To address this problem, we propose SAM-Flow, a source-anchored masked flow framework for localized training-free image editing. Instead of updating the whole latent representation, SAM-Flow first uses a scout image and token-grounded attention maps to localize the editable semantic regions. It then applies differential velocity updates only within these regions, while anchoring the remaining areas to the source-image latent trajectory. To further improve spatial stability and boundary naturalness, we introduce a time-varying source-anchored projection mechanism with dynamic soft masks, transition regions, and temporal mask accumulation. The proposed method is plug-and-play and can be integrated with mainstream flow-matching backbones such as Stable Diffusion 3 and FLUX without any fine-tuning. Extensive qualitative and quantitative experiments demonstrate that SAM-Flow achieves accurate semantic editing while significantly improving background preservation, providing a simple and general localized editing paradigm for training-free image editing. Code is available at: https://github.com/chwbob/Sam-Flow.

*   **研究背景**：近年来，基于扩散模型和流匹配（Flow Matching）的生成模型在文本到图像生成上取得了巨大成功。免训练编辑方法因其无需额外训练、直接利用预训练大模型的能力而备受关注。然而，主流方法（如图2的A、B两路径）都强加了对整张图像的分布映射，从根本上无法避免背景的连带改变。

### 2. 论文提出的方法论：核心思想、关键技术细节

*   **核心思想**：将编辑范式从**全局分布传输**转变为**局部化流编辑**。提出一种“源锚定掩码流”（Source-Anchored Masked Flow）框架，旨在识别可编辑的语义区域，仅在该区域内应用编辑力（差分速度），并严格地将背景区域锚定（anchor）到源图像的潜在轨迹上，确保其不变性。

*   **关键技术细节**：整个流程分为四个阶段（图3所示）。

    *   **阶段A：Token-Grounded Semantic Localization（基于令牌的语义定位）**
        *   **目的**：自动化地、精确地定位需要编辑的语义区域，解决目标物体与源物体空间位置不对齐的问题。
        *   **方法**：
            1.  **生成侦察图像（Scout Image）**：先用一个未经约束的编辑方法（如FlowEdit）生成一个初步编辑结果，作为探测目标语义可能出现的区域。
            2.  **提取注意力图**：对源图像和侦察图像应用相同噪声，从预训练模型的交叉注意力层提取对应源实体Token和目标实体Token的注意力图（M<sup>src</sup><sub>ti</sub> 和 M<sup>tar</sup><sub>ti</sub>）。
            3.  **构建联合掩码**：取两者的**并集** ( `M<sup>base</sup><sub>ti</sub> = max(M<sup>src</sup><sub>ti</sub>, M<sup>tar</sup><sub>ti</sub>)` ) 作为基础编辑支持区域。这个“侦察-联合”策略有效覆盖了物体替换、新增、删除及大形变等复杂场景下的完整编辑区域。

    *   **阶段B：Localized Differential Flow Editing（局部差分流编辑）**
        *   **目的**：继承FlowEdit免反演、免优化的优点，但只将编辑力作用于被掩码圈定的局部区域。
        *   **方法**：
            1.  **采样相关噪声**：为源分支和目标分支构造具有相关性的带噪潜变量（z<sup>src</sup><sub>ti,k</sub>, z<sup>tar</sup><sub>ti,k</sub>），减少随机噪声对编辑方向的影响。
            2.  **计算差分速度**：通过预训练模型预测源和目标速度场，计算差分速度 ∆v<sub>ti</sub> = v<sup>tar</sup> - v<sup>src</sup>。
            3.  **生成候选状态**：应用差分速度得到未约束的编辑候选状态 z<sup>edit</sup><sub>ti-1</sub>。这个状态仅作为下一步投影的“候选”，而非直接接受的状态，为后续源锚定操作提供了分离点。

    *   **阶段C：Time-Varying Source-Anchored Projection（时变源锚定投影）**
        *   **目的**：将候选编辑状态与源图像潜变量融合，确保编辑只发生在目标区域，同时实现自然、平滑的边界过渡。
        *   **方法**：
            1.  **动态三区软掩码**：根据时变的阈值和锐化系数，将基础注意力图动态划分为三个区域：
                *   **核心区（Core）**：主要编辑区域，允许大幅度的语义变化。
                *   **过渡区（Ring）**：核心区与背景之间的软带状区域，用于平滑过渡，避免硬边界。
                *   **背景区（Outer）**：严格保留源图像内容的区域。
            2.  **时域累积（Temporal Accumulation）**：对每个时间步生成的三区掩码进行逐元素取最大值累积，确保被标记为可编辑的区域在后续步骤中不会被突然移除，增强空间稳定性。
            3.  **源锚定投影**：最终的模型状态通过以下公式融合：
                *   `z<sup>model</sup><sub>ti-1</sub> = a<sup>model</sup><sub>ti</sub> + ρ * W<sup>soft</sup><sub>ti</sub> ⊙ (z<sup>edit</sup><sub>ti-1</sub> - a<sup>model</sup><sub>ti</sub>)` ，其中 `a<sup>model</sup>` 是源图像潜变量锚点。这意味着在核心区和过渡区，潜变量可以走向编辑候选状态；在背景区，潜变量被拉回源图像锚点。
            4.  **多状态组合**：最终用于解码的可见状态 `z<sup>vis</sup>` 由核心、过渡和背景三个状态加权组合而成，确保了最终输出是源锚定后的合成结果。

### 8. 不足与局限

*   **依赖侦察图像质量**：定位的准确性依赖于第一步生成的侦察图像的质量。如果侦察图像编辑结果不佳，可能会误导后续的掩码生成。
*   **计算开销**：生成侦察图像和多次采样计算差分速度（navg=8）增加了额外的计算成本，虽然比训练便宜，但比纯差分流方法（FlowEdit）耗时更多。
*   **动态掩码的局限性**：虽然动态，但掩码的构建仍然依赖固定的时间调度参数。在面对语义极其复杂或目标物体尺度变化极大的场景时，手动调整参数的通用性可能受挑战。
*   **非刚体/拓扑变换限制**：虽然支持“大形变”，但从提供的示例（如“熊变摩托车”）看，更多的还是语义替换，对于需要剧烈改变物体拓扑结构或姿态的编辑，效果可能不如专门的训练类模型。
