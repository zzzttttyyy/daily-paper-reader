## 摘要
从单张无约束照片重建逼真的3D面部头像具有挑战性：前馈式3D高斯泼溅（3DGS）模型在分布外输入上性能下降，而预训练的扩散模型能生成高保真图像但缺乏多视图一致性。我们观察到这些范式本质上是互补的：显式3D表示保证几何一致性，而2D扩散先验确保照片逼真性。基于此，我们提出Splatshot，一种无训练框架，直接在去噪过程中耦合这些表示。给定一个基础3DGS面部模型和一张参考图像，我们使用每步3D反馈循环联合去噪所有目标视图。在每个时间步，我们从含噪潜变量预测干净图像，将这些多视图预测重新拟合到3DGS，并将3DGS重渲染与2D预测之间的光度差异反向传播到噪声估计中。这引导采样轨迹朝向严格3D一致、身份保真的输出。在各种野外图像上的实验表明，Splatshot生成的3D头像具有优越的身份保留、照片逼真度和多视图一致性。

## Abstract
Reconstructing a photorealistic 3D face avatar from a single unconstrained photograph is challenging: feed-forward 3D Gaussian Splatting (3DGS) models degrade on out-of-distribution inputs, while pretrained diffusion models produce high-fidelity images but lack multi-view consistency. We observe that these paradigms are fundamentally complementary: explicit 3D representations guarantee geometric consistency, whereas 2D diffusion priors ensure photorealism. Building on this, we propose SplatShot, a training-free framework that couples these representations directly within the denoising process. Given a base 3DGS face model and a single reference image, we jointly denoise all target views using a per-step 3D feedback loop. At each timestep, we predict clean images from the noisy latents, refit the 3DGS to these multi-view predictions, and back-propagate the photometric discrepancy between the 3DGS re-renderings and 2D predictions into the noise estimate. This steers the sampling trajectory toward strictly 3D-coherent, identity-faithful outputs. Experiments on diverse in-the-wild images demonstrate that SplatShot produces 3D avatars with superior identity preservation, photorealism, and multi-view consistency.

## 2. 方法论

- **核心思想**：将预训练的基础3DGS面部模型与2D图像条件扩散模型（Stable Diffusion + IP-Adapter）在去噪每一步迭代耦合。通过每步的3D反馈，将扩散生成的多视图预测重新拟合到3DGS，再将重渲染与预测的光度误差反向传播到噪声估计，引导采样轨迹朝向3D一致。
- **关键技术细节**：
  - **初始化**：从300个预训练的NeRSemble基模型中选择与输入照片最匹配（基于DINO特征的加权相似度，优先匹配发型）的基础3DGS模型，渲染V=16个视图作为起点。
  - **部分噪声起点**：对初始潜变量加噪到起始时间步ts=0.6T（而非纯噪声），保留基础结构。
  - **每步循环**：
    1. **扩散生成**：独立预测每视图噪声，解码得到干净图像预测。
    2. **3DGS更新**：用预测图像梯度更新3DGS参数（首步1000次迭代，后续200次）。
    3. **噪声调整**：计算3DGS重渲染与预测图像的光度损失，梯度通过VAE解码器反向传播到噪声估计（公式5），调整噪声以增强3D一致性。
  - **噪声混合机制**：早期去噪步中，将预测噪声与初始添加的固定噪声混合（权重w=0.4），避免崩溃（图4）。

## 6. 主要结论与发现

- Splatshot在所有指标上达到或接近最优：**CV-CSIM最高（0.832）、AKD最低（0.92e-2）、FID最低（216）、CLIP-IQA最高（0.633）**，身份保真度CSIM第二（0.698），表明极好的多视图一致性和照片质量。
- 与传统方法相比，Splatshot无需特定优化，能处理遮挡、极端年龄、不同肤色、非正面姿态等挑战。
- 验证了显式3D表示与2D扩散先验在去噪过程中双向耦合的有效性：3D反馈强制视图一致，扩散提升逼真度。
- 噪声混合机制对于早期步骤稳定性至关重要。

## 7. 优点

- **无训练**：完全使用预训练模型，无需微调任何组件，即插即用。
- **泛化性强**：对分布外野生图像鲁棒，不受特定训练域限制。
- **严格3D一致**：最终输出为可渲染的3DGS，支持任意视角的逼真渲染。
- **身份保持好**：通过IP-Adapter实现强身份条件控制，同时通过基础模型提供几何锚定。
- **设计巧妙**：噪声混合与每步反馈循环简单有效，解决了早期崩溃问题。
- **全面的评估**：包括定性与定量、消融、公平性分析，结果可信。

## 8. 不足与局限

- **速度慢**：每个身份约3分钟，远慢于前馈方法（如GAGAvatar 107秒，FastAvatar 3秒），但优于部分优化方法（如Arc2Avatar 3.5小时）。
- **发型继承**：发型主要继承自基础模型（而非输入），因为身份编码器通常裁剪掉头发区域，且头发几何不稳定。若基础模型发型匹配差，输出会出现偏差。
- **视角限制**：基础模型基于NeRSemble训练，覆盖正面180度范围，未评估后视图生成。
- **配件处理**：能处理墨镜、耳环，但帽子处理困难（NeRSemble中缺少帽子示例，身份编码器可能将帽子视为头部一部分）。
- **计算成本较高**：定量全量测试需要大量算力（314个A100小时），可能限制大规模验证。
- **未进行用户研究或主观评价**：仅依赖自动指标，缺少人类评估。
- **其他**：未探索与其他扩散基础模型（如SDXL）的兼容性；加速方案（PeRFlow）因步数不足而失败，仍需进一步研究。
