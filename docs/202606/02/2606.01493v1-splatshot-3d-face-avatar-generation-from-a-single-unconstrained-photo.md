## 摘要
从单张非约束照片重建逼真的3D人脸头像具有挑战性：前馈3D高斯泼溅（3DGS）模型在分布外输入上退化，而预训练扩散模型生成的高保真图像缺乏多视角一致性。我们观察到这些范式本质上是互补的：显式3D表示保证几何一致性，而2D扩散先验确保照片真实感。基于此，我们提出SplatShot，一种无需训练、在去噪过程中直接耦合这些表示的框架。给定一个基础3DGS人脸模型和单张参考图像，我们通过逐步骤的3D反馈循环联合去噪所有目标视角。在每个时间步，我们从噪声潜变量预测干净图像，将这些多视角预测重新拟合到3DGS，并将3DGS重新渲染与2D预测之间的光度差异反向传播到噪声估计中。这将采样轨迹引导到严格3D一致且身份保真的输出。在多种野外图像上的实验表明，SplatShot生成的3D头像在身份保留、照片真实感和多视角一致性方面表现优越。

## Abstract
Reconstructing a photorealistic 3D face avatar from a single unconstrained photograph is challenging: feed-forward 3D Gaussian Splatting (3DGS) models degrade on out-of-distribution inputs, while pretrained diffusion models produce high-fidelity images but lack multi-view consistency. We observe that these paradigms are fundamentally complementary: explicit 3D representations guarantee geometric consistency, whereas 2D diffusion priors ensure photorealism. Building on this, we propose SplatShot, a training-free framework that couples these representations directly within the denoising process. Given a base 3DGS face model and a single reference image, we jointly denoise all target views using a per-step 3D feedback loop. At each timestep, we predict clean images from the noisy latents, refit the 3DGS to these multi-view predictions, and back-propagate the photometric discrepancy between the 3DGS re-renderings and 2D predictions into the noise estimate. This steers the sampling trajectory toward strictly 3D-coherent, identity-faithful outputs. Experiments on diverse in-the-wild images demonstrate that SplatShot produces 3D avatars with superior identity preservation, photorealism, and multi-view consistency.

# 二、论文提出的方法论

## 核心思想
- 给定一张输入照片，从一个预训练的基础3DGS人脸模型出发（提供结构初始化），在DDIM去噪的每一步中引入3D反馈循环：  
  1. 利用扩散模型独立预测每个视角的干净图像；  
  2. 用这些多视角预测重新拟合3DGS模型；  
  3. 计算3DGS重渲染与2D预测之间的光度损失，将该损失的梯度反向传播到噪声估计中，调整噪声，从而引导后续去噪轨迹。
- 通过这种迭代交互，扩散模型逐渐提升真实感，3DGS强制几何一致性，最终生成身份保真且多视角一致的3DGS头像。

## 关键技术细节
- **基础模型选择**：从NeRSemble数据集预训练的300个3DGS模型中，基于DINO ViT特征在头发、肤色、脸型三个区域的加权余弦相似度自动选择最佳匹配。
- **噪声混合机制**：早期去噪步骤中，直接用预测噪声得到的干净图像过于损坏。因此将预测噪声与初始添加的真实噪声按权重混合（\(w = 0.4\)），既保留梯度流又提供稳定预测，防止3DGS崩溃。
- **迭代三步骤**：  
  - Step 1: 每个视角v，用带IP-Adapter身份条件的UNet预测噪声 \(\epsilon^{t,v}_{\theta}\)，计算干净图像 \(\hat{x}^v_0\) 并解码得到 \(\hat{I}^v\)。  
  - Step 2: 用所有 \(\{\hat{I}^v\}\) 对3DGS模型M进行若干次梯度更新（首步1000次，后续200次）。  
  - Step 3: 计算光度损失 \(g = L_{\text{photo}}(\hat{I}^v, R(M, v))\)，通过链式法则求导 \(\partial g / \partial \epsilon^{t,v}_{\theta}\) 并更新噪声估计，然后执行DDIM采样步得到下一时刻潜变量。
- **算法流程**：详见论文Algorithm 1，简洁概括为：渲染初始视图 → 编码到潜空间 → 加噪至起始步 → 循环去噪，每步执行上述三步。

# 六、论文的主要结论与发现

1. **SplatShot在身份保真（CSIM 0.698）、多视角一致性（CV-CSIM 0.832）、感知质量（CLIP-IQA 0.633）上达到或超越所有对比方法**，尤其显著提升多视角一致性。

4. **方法在种族/年龄组间表现均衡**：CSIM在各分组间差异小，无明显偏见。
5. **速度与质量权衡**：虽慢于前馈方法（3min vs 几秒），但获得明显更优的几何一致性和真实感。

# 七、优点

- **训练免费（training-free）**：无需微调任何模型组件，直接利用现有预训练模型，易于部署。
- **巧妙融合互补范式**：显式3DGS和2D扩散先验在去噪过程中相互指导，同时提升几何一致性与照片真实感。
- **噪声混合策略**：简单而有效地解决了早期去噪步骤中预测不稳定问题，是技术亮点。
- **推理时可编辑性**：基础模型作为软锚点，可通过调整去噪强度s控制输出与基模的偏离程度。
- **人口公平性验证**：明确报告了不同群体表现，体现社会责任感。

# 八、不足与局限

- **速度慢**：约3分钟/身份，远长于前馈方法（几秒），不适合实时应用。作者提及尝试加速（PeRFlow）未成功。
- **几何锚定受限于基础模型**：基础模型来自NeRSemble（仅正面180°），因此未评估后视图生成能力；对帽子等配件处理差（训练数据缺失）。
- **发型继承问题**：由于身份编码器裁剪头发，发型主要继承自基础模型而非输入照片，不匹配时产生偏差。
- **实验规模有限**：主要定量仅100个FFHQ身份，虽然定性展示更多，但统计可靠性可进一步加强。
- **缺乏动态/动画能力**：输出静态3DGS模型，未涉及表情动画或驱动。
- **对极端条件鲁棒性不足**：如大角度遮挡、极端光照、帽子等仍有失败案例（附录图12）。
