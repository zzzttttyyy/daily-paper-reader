## 摘要
文本到图像扩散模型如Stable Diffusion能从文本生成高质量图像，但在推理时无法注入视觉引导（如草图、风格）而无需重新训练。现有方法要么需要计算昂贵的微调，要么依赖风格迁移技术，这可能导致与文本提示的语义不一致。我们提出了视觉概念融合（VCF），这是第一个在推理时无需任何概念特定训练即可同时处理图像和文本提示的双重条件方法。VCF通过将CLIP图像特征与文本嵌入空间对齐，实现了将视觉概念注入Stable Diffusion。VCF包含三个组件：（1）一个轻量级对齐器，使用InfoNCE和交叉注意力重建损失将图像标记映射到文本嵌入流形；（2）一种保留文本和视觉语义的融合策略；（3）一个可选的提示-噪声优化（PNO）模块，用于测试时优化。我们的实验表明，VCF成功地从参考图像传递了视觉属性，包括风格、构图和调色板，同时保持了对提示的遵循。定量结果显示，在文本对齐（CLIP分数）和视觉对应（LPIPS）之间存在权衡，VCF在参考保真度上优于基线。

## Abstract
Text-to-image diffusion models like Stable Diffusion generate high-quality images from text, but lack a way to inject visual guidance (e.g. sketches, styles) at inference without retraining. Existing methods either require computationally expensive fine-tuning or rely on style transfer techniques that risk semantic misalignment with textual prompts. We introduce Visual Concept Fusion (VCF), the first method offering dual conditioning on both an image and text prompt at inference time without any concept-specific training. VCF enables visual concept injection into Stable Diffusion by aligning CLIP image features with the text embedding space. VCF consists of three components: (1) a lightweight aligner that maps image tokens to the text embedding manifold using InfoNCE and cross-attention reconstruction losses, (2) a fusion strategy that preserves both textual and visual semantics, and (3) an optional Prompt-Noise Optimization (PNO) module for test-time refinement. Our experiments demonstrate that VCF successfully transfers visual attributes including style, composition, and color palette from reference images while maintaining prompt adherence. Quantitative results show a trade-off between text alignment (CLIP score) and visual correspondence (LPIPS), with VCF outperforming baselines in reference fidelity.

## 1. 核心问题与整体含义（研究动机和背景）

文本到图像扩散模型（如Stable Diffusion）能从文本提示生成高质量图像，但缺乏在推理阶段直接注入视觉引导（如草图、风格、参考图像）的能力。现有方法存在两个主要瓶颈：

- **微调方法**（如DreamBooth、StyleDrop、ControlNet）需要针对每个新概念或风格进行昂贵的重新训练或微调，计算资源消耗大，且依赖大量配对数据。
- **风格迁移方法**（如AdaIN）可能导致生成的图像与文本提示之间的语义错位，无法同时忠实地遵循文本内容和视觉参考。

这些方法均无法在推理时无需任何概念特定训练即可同时接受图像和文本的双重条件控制。论文旨在填补这一空白，提出**Visual Concept Fusion (VCF)**——首个无需任何概念特定训练即可在推理时实现图像和文本双重条件控制的方法。

## 2. 方法论：核心思想、关键技术细节

**整体框架**：VCF在冻结的Stable Diffusion模型之外，引入三个组件实现视觉概念注入。

### 2.1 图像到文本对齐（Image Aligner）

- **问题**：CLIP图像编码器的预投影token分布（p_image）与文本编码器的预投影token分布（p_text）不一致，直接将图像token注入文本条件模型会导致模态不匹配（通过KL散度量化）。
- **对齐器架构**：一个轻量级两层MLP（带LayerNorm和ReLU），参数约2.4M，是VCF中唯一从零训练的模块，扩散模型保持冻结。
- **损失函数**：
  - **全局对齐**：InfoNCE损失，拉近图像和文本token的平均嵌入。
  - **局部对齐（token级别）**：交叉注意力重建损失，用图像token重建文本token，保持token级结构。
  - 联合损失：L_align = λ * L_InfoNCE + L_attn，λ=0.2。
- **训练**：仅在COCO Captions的10%子集（约6万图像-标题对）上训练，不涉及扩散模型。

### 2.2 文本-图像融合（Text-Image Fusion）

对齐后的图像token（ˆI）与原始文本token（T）融合。论文对比三种策略：

- **Naive融合**：将图像token平均后与每个文本token线性混合（α=0.3控制强度）。效果差，产生噪声和语义不一致。
- **Concatenation融合（VCF主方法）**：将对齐的图像token直接拼接到文本token序列末尾，输入Stable Diffusion。保留各自语义，效果最佳。
- **交叉注意力融合**：文本token作为查询，对齐图像token作为键/值，生成融合表示后再与原文混合。效果比Naive好但不如拼接。

### 2.3 可选的提示-噪声优化（PNO）

- **作用**：测试时细化，进一步提升生成图像与参考图像的视觉对齐。
- **机制**：同时优化条件token T_final和初始噪声x_T，使最终生成图像与参考图像的CLIP嵌入余弦相似度最大化，同时加入正则化项（λ_reg=0.1）防止噪声退化。
- **步骤**：10-50步梯度更新，使用DDIM确定性采样（优化x_T即优化整个轨迹）。

## 3. 实验设计

### 数据集与场景

- **训练**：COCO Captions数据集，随机选取10%子集（约6万对），80/10/10划分训练/验证/测试。每个epoch随机抽取每张图像的一个描述。
- **评估**：给定参考图像和文本提示（如“A photo of a cat”、“A charming character emerging from the scene”、“a delicious pizza”等），生成图像，考察文本对齐和视觉对应。

### 基准方法

- **SDv2（text-only）**：只有文本提示，无视觉参考。
- **Naive融合**：直接将平均图像token线性混合到文本token（α=0.3）。
- **VCF（Ours）**：使用拼接融合，是否启用PNO可选。

- **CLIP Score**：生成图像与文本提示的CLIP余弦相似度（越高越好）。
- **LPIPS**：生成图像与参考图像的感知相似度（越低越好）。

## 6. 论文的主要结论与发现

- VCF成功在推理时将参考图像的风格、构图、颜色等视觉属性注入生成结果，同时较好地保留了文本提示的内容（如“cat”）。
- Naive融合几乎无效，生成图像接近text-only基线，表明简单的线性混合不足以弥合模态差距。
- 对齐器联合使用InfoNCE和交叉注意力损失至关重要：仅InfoNCE导致视觉引导微弱；仅交叉注意力导致提示遵循性差；两者结合取得最佳平衡。
- PNO可进一步改善文本仅或融合方法的视觉对齐和图像质量（减少噪声、增强色彩保真度）。
- VCF在抽象或模糊文本提示下表现尤为突出，能显著提升生成图像的语义连贯性和视觉细节。

## 7. 优点
